## 📊 P2P Agent Mesh — Architecture

### High-Level Overview

```
┌─────────────────────────────────────────────────────────┐
│                     Blind Relay                          │
│  (NAT traversal, routing only — no E2E content visible) │
└──────────────────────┬──────────────────────────────────┘
                       │
        ┌──────────────┼──────────────┐
        │              │              │
   ┌────▼────┐   ┌────▼────┐   ┌────▼────┐
   │ Agent A │   │ Agent B │   │ Agent C │
   │  (NAT)  │   │ (open)  │   │  (NAT)  │
   └────┬────┘   └────┬────┘   └────┬────┘
        │              │              │
   ┌────▼──────────────▼──────────────▼────┐
   │         TCP/P2P Transport              │
   │   (gossipsub topic-based routing)      │
   └───────────────────────────────────────┘
```

### Protocol Stack

```
┌────────────────────────────────────────────┐
│         Agent SDK (emit/listen/request)     │
├────────────────────────────────────────────┤
│         Coordination Layer (opt-in)         │
│   ┌──────────┬─────────────┬───────────┐   │
│   │ Micro-   │  Consumer   │ Exactly-  │   │
│   │ Raft     │  Groups     │ once      │   │
│   └──────────┴─────────────┴───────────┘   │
├────────────────────────────────────────────┤
│         Security Layer                      │
│   ┌──────────┬─────────────┬───────────┐   │
│   │ Ed25519  │  X25519      │ ChaCha20-  │   │
│   │ Auth     │  ECDH (PFS) │ Poly1305   │   │
│   └──────────┴─────────────┴───────────┘   │
├────────────────────────────────────────────┤
│         Transport Layer                     │
│   ┌──────────┬─────────────┬───────────┐   │
│   │ TCP/IP   │  Gossipsub  │ WAL Buffer│   │
│   │ Pub/Sub  │  DHT Peers  │ (SQLite)  │   │
│   └──────────┴─────────────┴───────────┘   │
└────────────────────────────────────────────┘
```

### Data Flow — P2P Message

```
Agent A                    Relay                    Agent B
   │                         │                         │
   │── HELLO (pubkey,nonce)─►│                         │
   │                         │── HELLO (pubkey,nonce)─►│
   │◄── CHALLENGE (signed)───│◄── CHALLENGE (signed)───│
   │── AUTH (signed)────────►│── AUTH (signed)────────►│
   │                         │                         │
   │══ E2E session (X25519+ChaCha20) ══════════════════│
   │                         │                         │
   │── ENCRYPTED_MESSAGE ───►│── ENCRYPTED_MESSAGE ───►│
   │                         │ (relay can't decrypt)    │
   │                         │                         │
   │◄── ACK ─────────────────│◄── ACK ─────────────────│
```

### Data Flow — Coordination (Raft)

```
   Leader                    Follower                 Follower
   │                         │                         │
   │── AppendEntries ───────►│── AppendEntries ───────►│
   │                         │                         │
   │◄── AppendEntries_OK ◄───│◄── AppendEntries_OK ◄───│
   │                         │                         │
   │ [commit index > N]      │                         │
   │                         │                         │
   │── commit broadcast ────►│── commit broadcast ────►│
   │                         │                         │
   │ deliver to consumer     │ deliver to consumer      │
   │ (ordered, deduplicated) │ (ordered, deduplicated)  │
```

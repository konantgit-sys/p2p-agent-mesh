# P2P Agent Mesh — Outreach Kit

## 🎯 Elevator Pitch (3 предложения)

> **Проблема:** AI-агенты сегодня привязаны к Kafka/Redis/REST — хрупко, дорого, централизованно.
> **Решение:** P2P Agent Mesh — децентрализованный pub/sub с E2E encryption, NAT traversal и опциональной exactly-once доставкой. Без блокчейна, без единой точки отказа.
> **CTA:** `git clone https://github.com/konantgit-sys/p2p-agent-mesh && docker-compose up -d` — 2 команды до mesh-сети.
